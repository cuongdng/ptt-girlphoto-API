### build on your server / 自己架在主機上

clone repo  / 下載 repo
> git clone https://github.com/yiyu0x/ptt-beauty-API.git && cd ptt-beauty-API

install package / 安裝所需套件
> pip3 install -r requirements.txt

run it / 執行
> python3 main.py
